How It Works
Algorithmic techniques powering PlagiaGuard
📝 Text Analysis Pipeline
1
Preprocessing
Lowercasing, punctuation removal, stopword filtering (174-word list), Porter stemming algorithm.
2
TF-IDF Vectorisation
Term Frequency–Inverse Document Frequency converts documents into weighted numeric vectors.
3
Cosine Similarity
Measures angular distance between document vectors — immune to length differences.
4
N-gram Jaccard Index
Overlapping character/word n-gram shingles catch paraphrasing and reordering.
💻 Code Analysis Pipeline
1
Comment & Format Strip
Removes single/multi-line comments, extra whitespace, and blank lines from source code.
2
Variable Normalisation
Identifier renaming detected and replaced with canonical VAR_N tokens — defeats obfuscation.
3
Token Sequence Matching
Lexical tokenisation then Longest Common Subsequence comparison on operator/keyword sequences.
4
Structural Fingerprinting
Control-flow hashes (loop nesting, conditional depth) compared via Winnowing algorithm.
